## ‚ú® What's Included

- **Vue 3** + **TypeScript** + **Inertia.js v2** - Modern SPA-like experience with SSR support
- **Tailwind CSS v4** - Utility-first styling with dark mode
- **Filament v4** - Beautiful admin panel out of the box
- **Laravel Fortify** - Complete authentication (login, register, 2FA, email verification)
- **Laravel Wayfinder** - Type-safe route generation for TypeScript
- **Reka UI** - Beautiful, accessible Vue components
- **Pest PHP v4** - Modern testing framework with browser testing support
- **Laravel Boost** - Development tools and MCP server integration
- Code quality tools: **Pint**, **Rector**, **PHPStan**, **ESLint**, **Prettier**
